# Portfolio 작업

폰트, 여백, 색상 등 기본값 설정

나눔고딕

Sora

Pretendard

## 퍼블리싱 작업

### HTML 레이아웃 제작

header 부분을 먼저 제작함

각 페이지의 세부 내용을 제작함

### CSS 작업

폰트나 여백사이즈는 기본적으로 vw를 이용하여 작업하였고

고정적인 값이 필요한 경우에는 rem을 이용하여 작업하였음

### JS 작업

스크롤을 내릴시 상단 헤더가 사라지는 효과 추가

배경색이 특정 구간을 지나면 바뀌는 효과 추가

Plugin 을 이용하여 마우스에 특정 이미지가 따라오는 효과 추가
