Problem & Solution

레이아웃 넘침

width 값을 px로 지정하였을때는 브라우저의 크기에 따라서 레이아웃이 넘치는 문제가 있었습니다. 그래서 width값을 %로 주어 브라우저의 환경이 달라졌을때도레이아웃이 넘치지않고 보일수 있도록 수정했습니다.

position 설정

carrers 영역의 이미지를 position을 이용하여 작업했는데 작업중에 원하는 위치로 설정이 되지 않는 문제가 있었습니다. 이유를 찾아보니, html 구조에서 기준으로 생각했던 영역이 부모관계가 아니라 다른 영역으로 설정되어 있었습니다. 그래서 html 구조를 부모자식 관계로 수정하고 position을 이용하여 이미지를 원하는 위치에 설정했습니다.

Review

처음으로 작업하는 반응형 사이트 제작이라 어떻게 진행해야 모든 환경에서 보이는 구조를 진행할수 있을까 고민했습니다. 고민해보다가 html구조는 pc환경을 기준으로 css작업은 mobile환경을 기준으로 작업을 진행했더니 진행이 잘 되었습니다.

여러가지 레이아웃이 넘치는 현상이나 처음에 프로토타입으로 제작한 느낌과 달라서 여러 수정을 거쳤지만 내가 작업한 사이트가 여러가지 모바일기기나 태블릿, 브라우저 에서도 제대로 구현이 되는걸 보고 만족스러운 작업이였습니다.
